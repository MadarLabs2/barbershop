# אימות התחברות – קוד ב-WhatsApp

האפליקציה משתמשת בקוד 4 ספרות שנשלח ללקוח ב-WhatsApp ממספר המספרה (0547267713).

## זרימת העבודה

1. **הלקוח** מזין מספר טלפון ולוחץ "בקש קוד"
2. **המספרה** רואה את הבקשה בדף הניהול ופותחת WhatsApp לשליחת הקוד ללקוח
3. **הלקוח** מקבל את הקוד, מזין אותו ומתחבר

## הגדרה

### 1. הרצת הטבלה ב-Supabase

היכנס ל-Supabase Dashboard → SQL Editor והרץ את הקובץ:
`supabase/migration_otp.sql`

### 2. דף שליחת הקודים (למספרה)

פתח את הקובץ `admin/otp-panel.html` בדפדפן (למשל על מחשב או טאבלט במספרה).

**בפעם הראשונה** תתבקש להזין:
- **Supabase URL** – מ-Supabase Dashboard → Settings → API
- **Supabase Anon Key** – מאותו מקום

שמור. מעכשיו הדף יציג בקשות קוד חדשות. לחץ על "שלח ב-WhatsApp" כדי לפתוח צ'אט עם הלקוח עם ההודעה המוכנה – חסר רק ללחוץ שליחה.

### 3. שמירת הדף

ניתן לשמור את `otp-panel.html` בסימניות או לפתוח אותו בכרטיסייה קבועה במכשיר במספרה.
