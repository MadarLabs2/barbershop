# דשבורד מנהל

## גישה

פתח את הקובץ `admin/manager-dashboard.html` בדפדפן (מחשב/טאבלט).

בפעם הראשונה הזן את פרטי Supabase (כמו ב-otp-panel).

## תוכן הדשבורד

- **ימים** – מסודרים לפי תאריך
- **תחת כל יום** – התורים עם שעה ושם הלקוח
- **לחיצה על תור** – נפתחת רשימה עם מספר טלפון שניתן ללחוץ עליו ולהתקשר

## הרצת המיגרציה

כדי לתמוך בטלפון ושם הלקוח, הרץ ב-Supabase SQL Editor:

```
supabase/migration_appointments.sql
```
